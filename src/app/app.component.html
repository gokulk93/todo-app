Quick Notes:

Overview

This is a simple To-do-list application. This app is useful for the people who do every work with a particular schedule systematically. So, this app allows users to manage their tasks easily. Aim To Do will help you stay focused and organized with :

  1. Daily planner
  2. Lists
  3. Goal Planner

Features

  1. Manage todo list in dashboard
  2. Shows the progress in cahrt format
  3. Download the todos in JSON format (export)
  4. Upload the downloaded JSON back to dashboard (import)
  5. Maintain quick notes across sessions

How to use

Add todo tasks on top of the page will be added to Todo column

When the tasks are in progress, simply drag and drop the respective tasks to Active column

Later if the tasks completed drag them to Done column

Any update or delete of a particular task can be done by edit or remove icon by hover over that tasks

Export or import the data in JSON format under the Features menu

Also to clear all the data, Clear data under Features menu will help to achive it

Qucik notes section will store your data for any quick references

NOTE:

This website does not send any of your data into the server. All your data are available in your browser session until unless it clears. If you clear the browser cookies/sessions all your data will be wiped out.
